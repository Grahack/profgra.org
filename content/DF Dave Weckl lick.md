[La video](https://youtu.be/yBligw-TWg8) de Darryn Farrugia.

```
RLR KLR KLL
```

Les `LR`` sont sur les toms, le `LL`` sur la caisse claire.

Compléter si besoin avec `KLR` puis crash à gauche ou `KLL` pour crash à droite.
